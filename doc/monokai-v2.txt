    cs.base = {
      dark = p.surface0,        -- "#19181a"
      black = p.surface1,       -- "#221f22"
      red = p.red,              -- "#ff6188"
      green = p.green,          -- "#a9dc76"
      yellow = p.yellow,        -- "#ffd866"
      blue = p.cyan,            -- "#78dce8"
      orange = p.orange,        -- "#fc9867"
      purple = p.purple,        -- "#ab9df2"
      cyan = p.cyan,            -- "#78dce8"
      white = p.foreground,     -- "#fcfcfa"
      neutral1 = p.neutral1,    -- "#c1c0c0"
      neutral2 = p.neutral2,    -- "#939293"
      neutral3 = p.neutral3,    -- "#727072"
      neutral4 = p.neutral4,    -- "#5b595c"
      neutral5 = p.neutral5,    -- "#403e41"
    }
<

- **Primary Colors:** `red`, `green`, `yellow`, `blue`, `orange`, `purple`, `cyan`, `white` are your vibrant accent colors.
- **Neutral Colors:** `neutral1` to `neutral5` are muted shades for less prominent elements.
- **Background Colors:** `dark` and `black` serve as your primary and secondary background colors.

CONFIGURATION                                      *monokai-v2-configuration*

There are 7 filter variants included in this plugin:

- |monokai-v2-classic| – Monokai Classic _(default for dark background)_
- |monokai-v2-light| – Monokai Light _(default for light background)_
- |monokai-v2-machine| – Monokai Machine
- |monokai-v2-octagon| – Monokai Octagon
- |monokai-v2-pro| – Monokai Pro
- |monokai-v2-ristretto| – Monokai Ristretto
- |monokai-v2-spectrum| – Monokai Spectrum

Example configuration:

>lua
    require("monokai-v2").setup({
      -- Basic settings
      transparent_background = false,
      terminal_colors = true,
      devicons = false,
      
      -- Syntax highlighting styles
      styles = {
        comment = { italic = false },
        keyword = { italic = false },
        type = { italic = false },
        storageclass = { italic = false },
        structure = { italic = false },
        parameter = { italic = false },
        annotation = { italic = false },
        tag_attribute = { italic = false },
      },
      
      -- Treesitter settings
      treesitter = {
        italic = false,
      },
      
      -- Filter selection
      filter = "classic", -- classic | light | machine | octagon | pro | ristretto | spectrum
      
      -- Day/night mode
      day_night = {
        enable = false,
        day_filter = "pro",
        night_filter = "spectrum",
      },
      
      -- Incremental search style
      inc_search = "background", -- underline | background
      
      -- Background clearing for floating windows
      background_clear = {
        "toggleterm",
        "telescope",
        "renamer",
        "notify",
      },
      
      -- Plugin-specific settings
      plugins = {
        bufferline = {
          underline_selected = false,
          underline_visible = false,
          underline_fill = false,
          bold = true,
        },
        indent_blankline = {
          context_highlight = "default", -- default | pro
          context_start_underline = false,
        },
      },
    })
<

Customization ~

Override specific highlight groups:

>lua
    require("monokai-v2").setup({
        override = function(c)
          return {
              Normal = { bg = "#000000" }
          }
        end
    })
<

Override palette colors:

>lua
    require("monokai-v2").setup({
        overridePalette = function(filter)
          return {
            surface0 = "#101014",
            surface1 = "#16161E",
            background = "#1A1B26",
            foreground = "#C0CAF5",
            red = "#f7768e",
            orange = "#7aa2f7",
            yellow = "#e0af68",
            green = "#9ece6a",
            cyan = "#0DB9D7",
            purple = "#9d7cd8",
            neutral1 = "#737aa2",
            neutral2 = "#787c99",
            neutral3 = "#363b54",
            neutral4 = "#363b54",
            neutral5 = "#16161e",
          }
        end
    })
<

Override the entire scheme:

>lua
    require("monokai-v2").setup({
        overrideScheme = function(cs, p, config, hp)
          local cs_override = {}
          local calc_bg = hp.blend(p.background, 0.75, '#000000')
    
          cs_override.editor = {
            background = calc_bg,
          }
          return cs_override
        end
    })
<

COMMANDS                                              *monokai-v2-commands*

- `MonokaiThemeSelect` - Launch interactive menu to choose theme filter
- `MonokaiChoose <filter>` - Direct filter selection (e.g., `MonokaiChoose pro`)

Available filters: classic, light, machine, octagon, pro, ristretto, spectrum
